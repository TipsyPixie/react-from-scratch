# React from Scratch
## Requirements
* Yarn 1.7+
* Node 9+
## Quick start
1. Install package manager Yarn
   1. https://yarnpkg.com
   2. Or with NPM
      ```bash
      npm -g install yarn
      ```
2. Install dependencies
   ```bash
   yarn install
   ```
3. Run nodemon
   ```bash
   yarn run dev
   ```
4. Happy hacking!
