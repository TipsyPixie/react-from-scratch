# React from Scratch
## Requirements
* Yarn 1.7+
* Node 9+
## Quick start
1. Rename .env.sample file to .env, and setup environment variables
   ```bash
   mv ./.env.sample ./.env
   ```
   ```bash
   NODE_ENV=development
   PORT=3000
   ...
   ```
1. Install package manager Yarn
   1. https://yarnpkg.com
   2. Or with NPM
      ```bash
      npm -g install yarn
      ```
1. Install dependencies
   ```bash
   yarn install
   ```
1. Run nodemon
   ```bash
   yarn run dev
   ```
1. Happy hacking!
